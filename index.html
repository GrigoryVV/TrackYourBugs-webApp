1. Enter to the cabinet
2. Creating a new project
3. Addition of bugs
4. Editing bugs
5. Editing user data

First step is to create an account or sign in with existing one. You can use test account if you don't want to create new one. email: test@mail.ru | password: test

When you entered the cabinet you can create a project in which you will track bugs. You can give a name and description, also the datetime and creator email will be saved automatically. After all, new project card will be added below so you can start working with it.

To enter the project bug menu you need to click on its card. Firstly, in bug menu you need to add a new bug. It's simmilar to the project creation proccess. The new bug is always created with the "open" status. You can change it in the future.

To edit a bug you need to click on the bug row in the table. In the open modal window you will be able to change a name, description and status. To save the changes puch the "update" button below.

To edit the user data you need to click on the "User Information" button in the left side menu. Here you can change a name, password, birthday and sex of the user. To save the changes puch the "update" button below.
